## Install script on Termux
```
$ pkg install python2
$ pkg install git
$ git clone https://github.com/anggaxd/pyprotect
```

## Run script
```
$ cd pyprotect
$ python2 enc.py -i (yourscript.py) -o (namefile.py) 
* Example : python2 enc.py -i /sdcard/y2.py -o namesc.py
```
